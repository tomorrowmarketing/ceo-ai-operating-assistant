## 프로젝트 한 줄 요약

대한민국 리드수집 광고대행사 **대표를 위한 개인용 AI 운영 비서**.
일반 챗봇이 아니라, 대표가 **매일 오전 5분 안에 운영 상황을 파악하는 운영 브리핑 대시보드**.

## 반드시 지킬 핵심 규칙

1. **광고주 = 계약 + 업무 + 커뮤니케이션 + 일정 + 재무 + 광고 성과.** 모든 것은 광고주 중심으로 정리.
2. AI는 **읽기·요약·우선순위·추천·초안 작성**만 한다. **대표 승인 없이 외부 행동 실행 금지.**
3. 1차 MVP는 **Mock 데이터만** 사용. 외부 API/인증/자동 실행 기능 없음.
4. **UI 문구는 전부 한국어**, 대표 관점. 일반 TODO 앱처럼 보이면 안 됨.
5. 상태 라벨은 **긴급 · 주의 · 정상 · 확인필요 · 승인필요** 5종.
6. **민감정보**(전체 전화번호/계좌/카드/주민번호/환자·사건 상세/원본 리드 DB/개인일정 상세)는
   저장 금지 또는 마스킹.
7. **과설계 금지.** 불필요한 의존성 추가 금지. 단순하고 리뷰 가능하게.

## 기술 스택

Next.js (App Router) · TypeScript · Tailwind CSS · Mock 데이터(TS/JSON).

## 디렉터리

- `src/app/` — 라우트(대시보드 `page.tsx`)
- `src/components/` — 영역별 대시보드 컴포넌트 (`ui/` 공통 프리미티브)
- `src/lib/types.ts` — 도메인 타입 단일 정의처
- `src/lib/mock/` — 가상 데이터 + `TODAY` 기준일
- `src/lib/briefing/` — 비즈니스 로직(순수 함수) + 일일 브리핑 조립
- `docs/` — 제품 문서 00~09

## 실행

```bash
npm install
npm run dev     # http://localhost:3000
npm run build
```

자세한 내용은 `CLAUDE.md` 와 `docs/` 를 참고한다.
